# VoltchZ Brasil — Infraestrutura de Recarga para Veículos Elétricos

Site institucional moderno e de alta conversão para a **VoltchZ Brasil**, empresa especializada em engenharia de infraestrutura para carregamento de veículos elétricos (VEs).

## 🚀 Tecnologias Utilizadas

- **Estrutura:** [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS
- **Interatividade:** JavaScript Vanilla (ES6+)
- **Design:** Focado em estética premium (Dark Mode, Glassmorphism, Grain Texture)
- **Acessibilidade & SEO:** Implementação completa seguindo boas práticas de mercado.

## ✨ Principais Funcionalidades

- **Design Premium:** Interface moderna com modo escuro, texturas cinematográficas e animações fluidas.
- **Carrossel Hero Inteligente:** Banner rotativo dinâmico que se adapta ao número de slides.
- **Galeria de Cases com Lightbox:** Visualização de projetos realizados com suporte a pausa no carrossel durante a inspeção.
- **Barra de Progresso e Scroll-spy:** Melhora a navegação e o entendimento da profundidade do conteúdo.
- **Foco em Conversão:** Botão de WhatsApp persistente e CTAs estratégicos em toda a jornada do usuário.
- **SEO Otimizado:** Meta tags Open Graph, Twitter Cards e estrutura semântica para melhor ranqueamento.
- **Acessibilidade:** Suporte a navegação por teclado e compatibilidade com leitores de tela (ARIA labels).

## 📂 Estrutura do Projeto

```text
├── IMAGENS/               # Ativos visuais (logos, banners, cases)
│   ├── CLIENTES/          # Fotos de projetos realizados
│   └── LOGOS-CLIENTES/    # Logos de parceiros e clientes
├── index.html             # Página principal (Landing Page)
├── sobre.html             # Página institucional detalhada
├── styles.css             # Estilos customizados e animações
├── scripts.js             # Lógica de interatividade e componentes
└── .gitignore             # Configurações de controle de versão
```

## 🛠️ Como Executar

O projeto é estático e não requer build process. Basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno.
